ABOUT WEBSITE

It is a Cloud-based web application that stores information about Residents in a Community.

The Resident data comprises information such as Resident's First Name, Last Name, Resident ID, Apartment Number, Street Name and their Photo.

This web application uses AWS Services such as EC2 to run Virtual Machine in the cloud, S3 buckets to store data, such as images, and Relational Database Cloud Services for database.
